A fun educational game built with **Python**, **Turtle Graphics**, and **Pandas**.  
Guess Ethiopian cities and see them appear on a map! Perfect for learning geography while practicing Python programming.

 #Features

- Interactive **Python Turtle** map
- Tracks correct guesses
- Prevents duplicate entries
- Exit option with score display
- Win screen when all cities are guessed
- Uses **Pandas** to read city data from CSV
